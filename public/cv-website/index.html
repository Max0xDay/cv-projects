About This Page

This page started off as a plain and uninteresting site that I left for months without any updates. It was originally built with Node.js, but I recently switched it to Deno and added a Three.js particle simulation in the background for some extra flair. I also finally added all my content, so now it’s a place for me to show off my background and what I’ve been up to.

It runs on my homeserver and I use Cloudflared tunneling (also called Zero Trust) to forward the page to the internet. This makes it super easy to access from anywhere without messing with router settings or exposing my network directly. Cloudflared handles the secure tunnel and keeps things simple.

Working on this taught me a lot about how static page loading works, and I ended up going down a rabbit hole about browser caching when some of my updates weren’t showing up. I’m still adding new stuff as I go, and I’m sure I’ll keep learning more as new opportunities come up.

Technical Stuff

The site is built with Deno and uses Three.js for the animated background. It's hosted on my homeserver and made public using Cloudflared tunneling, so I don’t have to open up any ports. I learned a lot about browser caching and static files while working on it. The code is on GitHub if you want to check it out or use it for your own site.
